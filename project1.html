Moshe's Insanity

Description Based on the puzzle Instant Insanity, Moshe's Insanity is a puzzle game that involves 8 cubes with a uniquely colored side. The goal is to assemble the 8 cubes into a 2x2x2 cube where all of the sides have unified colors.
I created a program that would allow users to input their own puzzle or generate a random one and attempt to solve it. The program could also determine, given a puzzle (whether randomly generated or inputted by a user), if the puzzle was solvable at all. All input and output was handled through the command line.

Timeline April 2020 - June 2020

Tech Stack Java, Maven
